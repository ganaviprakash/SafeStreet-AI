# 🛡️ SafeStreet AI

An elegant, modern safety awareness web application powered by simulated AI risk analysis. 

![Aesthetic Glassmorphism UI](https://via.placeholder.com/1200x600/f0e6ff/667eea?text=SafeStreet+AI+-+Aesthetic+Edition)

## ✨ Overview

SafeStreet AI allows users to evaluate the safety of various locations through community-driven data. With a massive visual upgrade, the application now sports a highly aesthetic **glassmorphism** design, fluid gradient backgrounds, and deeply satisfying micro-interactions. Simply input a location, select your personal safety rating, and watch the simulated AI instantly calculate a dynamic risk score!

## 🚀 Features

- **💎 Clean Glassmorphism UI:** Frosted glass cards, subtle borders, and soft shadows over an animated drifting blob background.
- **📍 Location Evaluation:** Submit locations mapped with a 1-5 initial rating block.
- **🤖 Simulated AI Risk Logic:** The robust Javascript logic calculates probability risk bands and outputs customized contextual warnings safely matched to your rating.
- **📊 Animated Results:** Includes dynamic percentage color-coded progress bars and fade-in result cards.
- **📜 Session History:** Persists all evaluations neatly via an in-memory chronological history feed to allow easy comparison.
- **📱 Fully Responsive:** The centered UI scales flawlessly on desktops, tablets, and mobile screens.

## 🛠 Tech Stack

Built deliberately without heavy front-end frameworks to demonstrate mastery over core web technologies:

- **HTML5:** Semantic architecture, accessible forms, and clear structure.
- **CSS3:** Advanced styling using fluid variables, `backdrop-filter` for glass panes, absolute positioning for animated blobs, and smooth transition properties.
- **JavaScript (ES6):** Vanilla JS for DOM manipulation, state management arrays, and event-driven AI simulation logic.
- **Google Fonts (Outfit):** Utilizes the modern *Outfit* typeface for a premium, startup-quality aesthetic.

## 💻 How to Run

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ganaviprakash/SafeStreet-AI.git
   ```
2. **Navigate into the project:**
   ```bash
   cd SafeStreet-AI
   ```
3. **Open the App:**
   Simply double-click `index.html` to open it in your browser of choice. (Alternatively, run it using a local dev server in VScode for the purest experience).
