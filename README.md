# SafeStreet-AI
SafeStreet AI is a smart safety awareness application that helps users evaluate the safety of locations based on community input.The platform allows users to enter a location and provide a safety rating, which is analyzed to generate a safety status such as Safe, Moderate, or Unsafe.

The goal is to build a crowdsourced safety network that helps people make informed decisions about their surroundings, with potential to evolve into an AI-driven system for predicting safety patterns and high-risk areas.
